### In plain English

The **mode** is the value that appears **most often** in a data set. Some lists have one mode, some have more than one, and some have no mode if everything appears equally.

### Example

Shoe sizes: $4, 5, 5, 5, 6, 7$. Size $5$ appears three times — more than any other — so the mode is $5$.

### Watch out

- The mode can be a number **or** a category name (like “red”).
- A list can be **bimodal** if two values tie for most frequent.
