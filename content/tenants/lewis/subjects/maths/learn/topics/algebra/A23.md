### What this is about

A **[sequence](../glossary/sequence.md)** is an ordered list of numbers, often following a rule. You might **continue** a pattern, find a **missing term**, or describe how each term relates to the one before (**term-to-term rule**).

Some sequences **add or subtract** the same amount each time — these are **linear** (arithmetic) patterns. Others **multiply** by the same factor each time — a **geometric** style pattern. Some go up, some go down; some use doubling instead of adding.

Spotting the rule helps you predict far-ahead terms without writing out every step. Sequences appear in stairs, savings plans, and tables in science — anywhere values change in a regular way.

### Words to know

- **Term** — one number in the sequence; “1st term”, “2nd term”, and so on.
- **Term-to-term rule** — what you do to get from one term to the next, e.g. “add 4”.
- **Position** — which place in the list (1st, 2nd, 3rd, …).
- **Common difference** — the fixed amount added or subtracted in a linear sequence.
- **Common ratio** — the fixed multiplier in a multiply-each-time sequence.

### Worked example 1

Write the next three terms of 7, 11, 15, 19, …

The rule is **add 4** each time.

Next terms: 23, 27, 31.

### Worked example 2

A sequence starts at 3 and **doubles** each time. Write the first five terms.

$3 \to 6 \to 12 \to 24 \to 48$.

**Answer:** 3, 6, 12, 24, 48.

### Tips for practice

- Write the **differences** between neighbours — if they stay constant, you are adding each time.
- If differences are not constant, try **ratios** (divide a term by the one before).
- Say the rule in words before you use algebra — “start at 5, subtract 2” is clearer than guessing.
- Check your rule on an early term you did not use to build it.
