### What this is about

Algebra uses three kinds of writing, and they behave differently.

An **[expression](../glossary/expression.md)** is a calculation **without** an equals sign. “A number plus 5” might be written $n + 5$. You can simplify it or find its value if you know $n$, but it is not “solved” — there is no single answer unless someone tells you $n$.

An **[equation](../glossary/equation.md)** **does** have an equals sign and is usually **true for some values** only. $x + 2 = 9$ is true when $x = 7$ and false when $x = 3$. Solving means finding the value(s) that make it true.

An **identity** is true **for every** allowed value of the letter. $x + x$ and $2x$ always match, so $x + x ≡ 2x$ is an identity (≡ means “identical to”). You are not hunting one missing number — you are showing two forms always agree.

Knowing which type you have tells you what to do next: simplify an expression, solve an equation, or prove an identity.

### Words to know

- **[Expression](../glossary/expression.md)** — symbols and numbers with operations; no “=”.
- **[Equation](../glossary/equation.md)** — two things set equal; often one unknown to find.
- **Identity** — always true, for all values (e.g. $3(x + 2) ≡ 3x + 6$).
- **Term** — parts added together, like $4x$ and $-7$.
- **Like terms** — same letter part, e.g. $2x$ and $5x$.
- **Coefficient** — the number in front of the letter in a term.

### Worked example 1

Write an expression for “7 less than a number $t$”.

“Less than” reverses order: start with $t$, subtract 7.

**Answer:** $t - 7$ (not $7 - t$ for this wording).

### Worked example 2

Is $2(x + 4)$ the same as $2x + 4$ for every $x$?

Expand the left: $2(x + 4) = 2x + 8$. That is $2x + 8$, not $2x + 4$.

So they are **not** an identity — try $x = 1$: left gives 10, right gives 6.

### Tips for practice

- No equals sign → expression. Equals sign → equation or identity.
- Test an identity with two easy values (0 and 1) if you are unsure.
- “Twice a number” → $2n$; “number doubled and then 3 added” → $2n + 3$.
- Like terms can be combined; unlike terms (different letters) cannot.
