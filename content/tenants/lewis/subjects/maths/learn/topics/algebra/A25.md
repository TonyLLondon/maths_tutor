### What this is about

For a **linear sequence** (same amount added or subtracted each time), you can write a **formula for the nth term** — a rule that jumps straight to the 100th term without listing 99 steps in between. If the sequence goes up by 4 each time, the $n$th term often looks like **$4n + \text{something}$**.

To find the formula, compare the sequence to the **times table** of the common difference. Sequence 5, 9, 13, 17, … adds 4; the table $4n$ gives 4, 8, 12, 16, … — each term is **1 more** than the table, so the nth term is **$4n + 1$**.

The nth term links [sequences](../glossary/sequence.md) to algebra: $n$ is the **position** (1, 2, 3, …). Substitute $n = 10$ to get the 10th term, or set the formula equal to a target to find which position matches a given value.

### Words to know

- **nth term** — a formula for term number $n$, often written $T_n$ or “nth term = …”.
- **Common difference** — add/subtract amount between neighbours in a linear sequence.
- **Position $n$** — 1 for the first term, 2 for the second, and so on.
- **Linear sequence** — constant difference between consecutive terms.
- **Offset** — the extra number added after multiplying $n$ by the difference, e.g. $+1$ in $4n + 1$.

### Worked example 1

Find the nth term of 6, 10, 14, 18, …

Common difference **4**. Compare to $4n$: 4, 8, 12, 16 — each sequence term is **2 more**.

**nth term:** $4n + 2$.

Check $n = 3$: $4(3) + 2 = 14$ ✓

### Worked example 2

The nth term of a sequence is $5n - 3$. Find the 8th term and the 1st term.

8th: $5(8) - 3 = 37$.

1st: $5(1) - 3 = 2$.

**Answer:** 8th term 37; 1st term 2.

### Tips for practice

- Find the **difference** first — that number usually multiplies $n$.
- Build a small table: $n = 1, 2, 3$ and your sequence side by side with $dn$ to spot the offset.
- For decreasing sequences, the formula can look like $-3n + 20$ — the coefficient of $n$ is negative.
- Always test the nth term on the **1st term** ($n = 1$) before you trust it.
