### What this is about

Besides “add 3 each time”, maths uses **named families** of [sequences](../glossary/sequence.md) that show up again and again. **Square numbers** are $1, 4, 9, 16, …$ — dots that form perfect squares ($1^2, 2^2, 3^2, …$). **Triangular numbers** grow by adding $1, 2, 3, 4, …$ to the running total: 1, 3, 6, 10, 15, …

**Cube numbers** come from $1^3, 2^3, 3^3, …$ — useful for volume patterns. **Fibonacci** starts 1, 1, then each new term is the **sum of the two before**: 1, 1, 2, 3, 5, 8, … Doubling sequences ($2, 4, 8, 16, …$) are another special type with a multiply-by-2 rule.

Recognising these families saves time. A question might ask for the 6th triangular number or whether 40 is a square number — you know which list to use.

### Words to know

- **Square number** — result of $n \times n$, e.g. 25 = $5^2$.
- **Cube number** — result of $n \times n \times n$, e.g. 27 = $3^3$.
- **Triangular number** — dots in a triangle; add consecutive whole numbers from 1.
- **Fibonacci sequence** — each term after the start is the sum of the two previous terms.
- **Perfect square / cube** — a whole number that is exactly a square or cube of an integer.

### Worked example 1

Write the first five square numbers and the 5th square number.

Squares: $1^2=1$, $2^2=4$, $3^2=9$, $4^2=16$, $5^2=25$.

The **5th square number** is 25.

### Worked example 2

Find the next two Fibonacci terms after 5, 8, 13.

Next: $8 + 13 = 21$. Then $13 + 21 = 34$.

**Answer:** 21, 34.

### Tips for practice

- Square numbers skip odd gaps — 7 is **not** a square; check if $\sqrt{n}$ is a whole number.
- Triangular numbers: the 4th is $1+2+3+4 = 10$ — add from 1 up to the position number.
- Fibonacci needs **two** starting terms; do not assume “add 1” unless the question says so.
- Keep square, cube, and “double each time” lists separate — they look similar at the start but diverge fast.
