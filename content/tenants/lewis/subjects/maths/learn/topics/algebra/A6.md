### What this is about

Both [equations](../glossary/equation.md) and identities use an equals sign, but they ask different questions.

An **equation** is like a balance that is level **only for certain values**. $3x = 12$ works when $x = 4$ and fails when $x = 2$. Solving finds those special values.

An **identity** is true **no matter what** (for all values in the domain). $x + x = 2x$ whether $x$ is 0, 7, or $-3$. You **show** an identity by simplifying both sides to the same form, not by finding one answer.

In practice you **test** a value in an equation to see if it works. For an identity, testing one value is **not enough** — you must simplify algebraically or test several values and still explain why it is always true.

### Words to know

- **[Equation](../glossary/equation.md)** — true for some values; solve to find them.
- **Identity** — true for every allowed value of the variable.
- **Verify / check** — substitute a number to see if an equation holds.
- **Simplify** — rewrite each side to compare forms.
- **Counter-example** — one value that shows two expressions are **not** identical.

### Worked example 1

Is $x + 5 = 8$ an equation or something true for all $x$?

It fails for $x = 0$ (left 5, right 8) and works for $x = 3$. So it is an **equation** with solution $x = 3$.

### Worked example 2

Show that $(x + 2)(x + 2)$ is identical to $x^2 + 4x + 4$.

Expand the left: $(x + 2)(x + 2) = x^2 + 2x + 2x + 4 = x^2 + 4x + 4$.

Both sides match — this is an **identity**.

### Tips for practice

- One value matching does **not** prove an identity.
- One value **not** matching proves two expressions are **not** an identity.
- Expand brackets fully before you decide if two sides match.
- Equations often have one solution; identities have infinitely many “solutions” (all values work).
