### What this is about

A **function** is a rule that takes an **input**, does something to it, and gives an **output**. You may see it as a function machine or as notation like $f(x) = 2x + 1$. The name $f$ labels the rule; $f(4)$ means “apply the rule to input 4”.

The input is often $x$; the output is $f(x)$ or $y$. Each input should give **one** clear output. Doubling a number always gives one result — that is a function. “Square root of $x$” as a function usually means the **positive** root in school maths.

You find outputs by [substitution](../glossary/substitute.md). You find inputs by **working backwards** with inverse operations — the same idea as missing numbers in earlier topics.

Functions describe rules in games, coding, science, and graphs: every $x$ on a graph paired with one $y$ from a formula.

### Words to know

- **Function** — a rule mapping each input to one output.
- **Input** — the value you start with (often $x$).
- **Output** — the result after the rule (often $f(x)$ or $y$).
- **Function notation** — writing $f(x) = …$ or $g(t) = …$.
- **Composite** — applying one function after another (output of first becomes input of second).
- **Inverse function** — undoes the original function (when it exists).

### Worked example 1

If $f(x) = x + 6$, find $f(2)$ and $f(-1)$.

$f(2) = 2 + 6 = 8$. $f(-1) = -1 + 6 = 5$.

**Answers:** 8 and 5.

### Worked example 2

$f(x) = 3x - 2$. If the output is 13, find the input.

Set $3x - 2 = 13$. Add 2: $3x = 15$. Divide by 3: $x = 5$.

**Answer:** input is 5. Check: $3(5) - 2 = 13$ ✓

### Tips for practice

- $f(3)$ always means substitute 3 for $x$ in the rule — not “$f$ times 3”.
- For “output is … find input”, write an [equation](../glossary/equation.md) and solve.
- Chained machines: do step one, then feed the result into step two.
- Keep track of which letter the function uses ($t$ for time, etc.).
