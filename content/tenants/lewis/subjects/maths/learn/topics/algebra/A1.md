### What this is about

Before algebra uses lots of letters, you learn the same ideas with a **missing number** in a box. Something plus 2 equals 7 — what goes in the box? You use the opposite operation: subtract 2 from 7 to get 5.

A **letter** like $n$ or $x$ is just a tidy way to write “the missing number”. The sentence $n + 3 = 7$ means the same as □ + 3 = 7.

**Function machines** are another way to think about it. A number goes **in**, something happens (add, multiply, and so on), and a new number comes **out**. You can also work **backwards** from the output to find the input — that is the same skill as finding a missing number.

You use these ideas whenever a game score, a height, or a price is hidden and you need to undo steps to find it.

### Words to know

- **Missing number** — the value that makes a number sentence true.
- **Variable** — a letter that stands for a number (often $x$, $n$, or $y$).
- **Input** — the number you put into a function machine or rule.
- **Output** — the number that comes out after the rule runs.
- **Inverse operation** — the opposite that undoes a step (add ↔ subtract, multiply ↔ divide).
- **Number sentence** — something like $4 + □ = 9$ or $2 × n = 12$.
- **Function machine** — a picture or story: input → do something → output.

### Worked example 1

Find $n$ if $n + 8 = 15$.

Think: “What plus 8 makes 15?” Using the inverse, subtract 8 from both sides mentally: $15 - 8 = 7$.

**Answer:** $n = 7$. Check: $7 + 8 = 15$ ✓

### Worked example 2

A machine does **× 4**, then **+ 3**. The input is 6. Find the output.

First step: $6 × 4 = 24$. Second step: $24 + 3 = 27$.

**Answer:** output is 27.

Now work backwards: the output is 27 after the same machine. Before + 3, you had $27 - 3 = 24$. Before × 4, you had $24 ÷ 4 = 6$.

**Input was 6** — backwards steps undo forwards steps.

### Tips for practice

- For □ + a = b, think $b - a$. For □ − a = b, think $b + a$.
- With two steps in a machine, do them **in order** forwards; undo them **in reverse order** backwards.
- Multiplying by 0 gives 0 — the missing number in □ × 0 = 0 is **not** always one special number.
- Say the inverse out loud: “I added 5, so I subtract 5 to go back.”
