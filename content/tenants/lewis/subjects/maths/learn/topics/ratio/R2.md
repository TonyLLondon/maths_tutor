## What this is about

A **scale factor** tells you how many times bigger or smaller a copy of a shape is compared with the original. Scale factor 3 means every length is multiplied by 3. Scale factor $\frac{1}{2}$ means each length is halved.

Maps and models use scale too: 1 cm on the map might stand for 5 km in real life. That is a [ratio](../../glossary/ratio.md) between drawing and reality. Scale factors help you read maps, build models, and resize diagrams while keeping the same shape.

When the scale factor is between 0 and 1, the image is a **reduction**. When it is greater than 1, it is an **enlargement**. Scale factor 1 means same size.

## Words to know

- **[Scale factor](../../glossary/scale-factor.md)** — the number you multiply each length by.
- **Enlarge / reduce** — make bigger (factor > 1) or smaller (factor < 1).
- **Map scale** — how map distance links to real distance (e.g. 1 cm : 2 km).
- **Similar** — same shape, possibly different size.
- **[Ratio](../../glossary/ratio.md)** — often written as 1 : n for “1 unit on the model to n units in real life”.

## Worked example 1

A line is 7 cm long. It is enlarged by scale factor 4. Find the new length.

New length = $7 \times 4 = 28$ cm.

**Answer:** 28 cm.

## Worked example 2

A map uses 1 cm for 4 km. How far in real life does 6 cm on the map represent?

Each cm is 4 km, so $6 \times 4 = 24$ km.

**Answer:** 24 km.

## Tips for practice

- Scale factor applies to **lengths**, one edge at a time — width, height, and radius all use the same factor.
- From ratio 1 : 100, the scale factor from drawing to real is **100** (drawing × 100 = real).
- To go from real distance to map distance, **divide** by the km-per-cm (or m-per-cm) value.
- Draw a tiny sketch with labels when a word problem mixes map cm and real km.
- A scale factor of $\frac{1}{3}$ means multiply lengths by $\frac{1}{3}$, not subtract 3.
