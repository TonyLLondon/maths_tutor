### What this is about

When a value is **[rounded](../glossary/rounding.md)** or measured to a given accuracy, the **true** value could be a little higher or lower. The **error interval** (bounds) shows all values that would round to the stated number.

“A length is 8 cm to the nearest cm” means the actual length is from 7.5 cm up to (but not including) 8.5 cm. We write $7.5 \le \text{length} < 8.5$.

Half the unit of rounding goes each side: nearest 10 → ±5; nearest 0.1 → ±0.05.

This idea matters for science measurements and for the next step — limits of accuracy in calculations.

Always check **which way** you round (5 rounds up in school maths for positive numbers).

### Words to know

- **[Rounding](../glossary/rounding.md)** — replace with a nearby value to a set accuracy.
- **Error interval / bounds** — range of true values.
- **Upper bound** — largest possible true value (just below the top of the interval).
- **Lower bound** — smallest possible true value.
- **Accuracy** — how fine the measurement is (nearest cm vs nearest mm).

### Worked example 1

A mass is 40 kg, correct to the nearest 10 kg. Write the error interval.

Nearest 10 → half-width 5 kg.

**$35 \le \text{mass} < 45$** kg.

### Worked example 2

A number is 6.2, correct to 1 decimal place. Write the error interval.

Half of 0.1 is 0.05.

**$6.15 \le \text{number} < 6.25$**.

For 8 cm to the nearest cm: $7.5 \le L < 8.5$.

For 300 to the nearest 100: $250 \le n < 350$.

### Tips for practice

- Write inequalities with the **smaller number included**, top **not** included (unless your teacher uses a different convention — match the worksheet).
- “Nearest 5 cm” → half-width 2.5 cm.
- The stated rounded value must lie **inside** your interval.
- Draw a tiny number line with the rounding points (7.5, 8.5) to see who rounds to 8.
- “Correct to 2 decimal places” means the hundredths digit is trusted; bounds extend ±0.005.
- Trailing zeros can show accuracy: 7.50 to 2 d.p. is not the same story as 7.5 to 1 d.p.
