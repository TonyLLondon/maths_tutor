### What this is about

Once you can write moves as **vectors**, you can **combine** them. A column vector $\binom{x}{y}$ is not just for one slide—it behaves like a number you can add, subtract, and scale.

**Adding** vectors means doing one move after another: add the top numbers for total horizontal shift, add the bottom numbers for total vertical shift. **Subtracting** finds the vector from one point to another. **Multiplying by a number** (a scalar) stretches the move: $2\binom{3}{1} = \binom{6}{2}$ means the same direction but twice as far.

Negative vectors flip direction: $-\binom{4}{2} = \binom{-4}{-2}$. This helps with paths that go there and back, or with finding a missing step in a journey.

Vector arithmetic is the language of forces in science, combined movements on grids, and harder geometry proofs later on.

### Words to know

- **Vector** — directed move stored as $\binom{x}{y}$.
- **Scalar** — an ordinary number that scales a vector (e.g. 3 or $-2$).
- **Component** — the horizontal or vertical part of a vector.
- **Resultant** — vector sum when two moves combine.
- **Column vector** — vertical stack: top = $x$, bottom = $y$.
- **Zero vector** — $\binom{0}{0}$; adding it changes nothing.
- **Equal vectors** — same components; same length and direction.
- **[Coordinates](../../glossary/coordinates.md)** — linked when a vector starts at the origin and ends at $(x, y)$.

### Worked example 1

**Task:** $\mathbf{a} = \binom{3}{-2}$ and $\mathbf{b} = \binom{-1}{4}$. Find $\mathbf{a} + \mathbf{b}$ and $\mathbf{a} - \mathbf{b}$.

**Steps:**

1. Add: $\binom{3 + (-1)}{-2 + 4} = \binom{2}{2}$.
2. Subtract: $\binom{3 - (-1)}{-2 - 4} = \binom{4}{-6}$.

Add and subtract **component by component**—do not mix top with bottom.

### Worked example 2

**Task:** $\mathbf{v} = \binom{6}{9}$. Find $-\frac{2}{3}\mathbf{v}$.

**Steps:**

1. Multiply each part: $-\frac{2}{3} \times 6 = -4$, and $-\frac{2}{3} \times 9 = -6$.
2. Result: **$\binom{-4}{-6}$**.

The vector points the opposite way and is two-thirds as long as $\mathbf{v}$ in each direction.

### Tips for practice

- Brackets help when subtracting negatives: $3 - (-1) = 4$.
- $\mathbf{a} - \mathbf{a} = \binom{0}{0}$—same vector subtracted cancels.
- For $k\mathbf{a}$, both components multiply by $k$, including the sign of $k$.
- Sketch arrows on a grid to see if your sum “looks” like two steps chained together.
- Order matters for subtraction: $\mathbf{a} - \mathbf{b}$ is not the same as $\mathbf{b} - \mathbf{a}$.
