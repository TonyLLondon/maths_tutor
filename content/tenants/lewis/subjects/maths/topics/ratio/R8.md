## Getting started

1. Complete: if x = 1 and y = 2, then x : y = ___.
2. If each bag holds 3 apples, how many apples are in 2 bags?
3. Complete the rule: multiply x by 2 to get y, so y = ___x.
4. Using y = 2x, find y when x = 3.
5. Using y = 4x, find y when x = 2.
6. In the rule y = 5x, what number multiplies x?
7. A table has x = 1, 2, 3 and y = 3, 6, 9. Complete the rule y = ___x.
8. Does the rule y = 2x give y = 0 when x = 0?
9. Using y = 3x, find y when x = 4.
10. Using y = 6x, find x when y = 18.
11. If x = 2 and y = 10, complete the rule y = ___ x.
12. A table has x = 0, 1, 2 and y = 0, 4, 8. Complete the rule y = ___ x.

## Reasoning (AO2)

24. Mia says every ratio table represents a straight-line graph through the origin. Is she correct? Explain why.

25. **Always, sometimes, or never?** A function of the form y = kx passes through the point (0, 0).

26. **True or false?** If x : y is always 1 : 3, then y is directly proportional to x. Explain.

27. Priya has a table where x : y is 2 : 5 for some rows but 2 : 6 for others. Can this data be described by a single rule y = kx? Explain.

28. Explain how you can find k in y = kx just from one pair of values (x, y), where x ≠ 0.

29. Sam says a conversion graph for pounds to euros must be a straight line through the origin, since £0 = €0. Is Sam correct?

30. Leo says that if you double x in y = kx, then y also doubles. Is Leo correct? Explain using the ratio.
