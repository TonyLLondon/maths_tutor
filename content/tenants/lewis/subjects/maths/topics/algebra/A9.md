## Getting started

1. For $y = x$, find $y$ when $x = 1$.
2. For $y = x$, find $y$ when $x = 3$.
3. For $y = x + 1$, find $y$ when $x = 2$.
4. For $y = x + 2$, find $y$ when $x = 1$.
5. For $y = 2x$, find $y$ when $x = 2$.
6. For $y = 3x$, find $y$ when $x = 2$.
7. On the grid, which point lies on $y = x$: $(2, 2)$ or $(2, 3)$?
8. On the grid, which point lies on $y = x + 1$: $(1, 1)$ or $(1, 2)$?

## Problem (AO3)
