## Problem (AO3)

## More reasoning (AO2)

93. **True or false?** A function must give exactly one output for each input. Explain.

94. **Always, sometimes, or never?** $f(x) = x + 1$ and $f(a) = a + 1$ mean the same process.

97. **Always, sometimes, or never?** $f(x) = 2x$ doubles every input.

98. Nina says $f(0) = 0$ for every function. Give a counter-example.

99. **True or false?** The input is called the independent variable in many graphs.

101. **True or false?** $f(x) = 5$ is a constant function.

102. **Always, sometimes, or never?** If $f(1) = 4$ and $f(2) = 8$, the function might be $f(x) = 4x$.

103. Omar says $f(x) = \dfrac{1}{x}$ works for $x = 0$. Explain.

104. **True or false?** $f(a + b) = f(a) + f(b)$ for $f(x) = 2x$ with $a = 3$, $b = 4$.

105. **Always, sometimes, or never?** $f(x) = |x|$ outputs a non-negative number.

106. **True or false?** A function machine diagram shows the same information as $f(x)$ notation.

112. **Always, sometimes, or never?** $f(x) = x^2$ is increasing for all real inputs.
