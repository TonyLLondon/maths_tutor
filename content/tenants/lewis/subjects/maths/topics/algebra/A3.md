## Getting started

1. Write the missing number: 3 + □ = 5.
2. Write the missing number: □ + 4 = 7.
3. Which letter is used in $x + 2$?
4. How many lots of $x$ are in $2x$?
5. If $x = 3$, what is $x + 1$?
6. If $y = 2$, what is $y + y$?
7. Complete the true number sentence: 6 = □ + 1.
8. Is $2 + 3 = 5$ true or false?

## Reasoning (AO2)

23. Priya says $2x + 3$ is an equation because it has an $x$ in it. Explain why Priya is wrong.

24. **Always, sometimes, or never?** An expression can be solved to find a single value of $x$.

25. Sam writes the identity $2(x + 3) \equiv 2x + 6$. Explain what the $\equiv$ symbol means and why it is used here instead of $=$.

26. Explain the difference between an **expression**, an **equation**, and a **formula**, using one example of each.

27. **True or false?** $x + x + x = 3x$ is true for every possible value of $x$, so it could be written as an identity.

28. Leo says the perimeter formula $P = 4s$ for a square is an equation. Explain why it is better described as a **formula**.

29. Compare $5x + 2$ and $5x + 2 = 17$. What can you find from the second one that you cannot find from the first?

30. Mo says "an equation always has an equals sign, so $2x + 3$ must be an equation." What is wrong with Mo's reasoning?

## More reasoning (AO2)

93. **True or false?** Every equation has exactly one solution. Give a counter-example.

94. **Always, sometimes, or never?** An identity is true for every allowed value of the variable.

95. Priya writes $3x$ as an equation. Explain the mistake.

96. **True or false?** $2x + 3$ can be evaluated but not “solved”. Explain.

97. **Always, sometimes, or never?** $x + x = 2x$ is an identity.

98. Sam says $x = 5$ is an expression. Correct him.

99. **True or false?** A formula is a special type of equation with several letters.

100. **Always, sometimes, or never?** $3x + 1 = 3x + 2$ has no solutions.

101. **True or false?** The word “is” in “$x$ is 5 more than 3” suggests an equation $x = 8$.

102. **Always, sometimes, or never?** Identities can be checked by substituting one value.

103. Leo writes $2x + 3 \equiv 11$. Explain why $\equiv$ is wrong here.

104. **True or false?** $x^2 + x^2 = 2x^2$ is an identity.

105. **Always, sometimes, or never?** An expression must contain a variable.

106. **True or false?** $5x + 2 = 2 + 5x$ is an identity.

107. **Always, sometimes, or never?** Equations can be true for some values and false for others.

108. Nina says identities are “always equations”. Explain why that is confusing.

109. **True or false?** $3(x + 2)$ and $3x + 2$ are different expressions.

110. **Always, sometimes, or never?** $x + 1 = 1 + x$ is true for every number $x$.

111. **True or false?** A coefficient can be negative.

112. **Always, sometimes, or never?** $2x = x$ has only the solution $x = 0$ if $x$ is a number.

113. **True or false?** $x + 0 = x$ is an identity.

114. **Always, sometimes, or never?** “Expression” and “term” mean the same thing.

115. **True or false?** $x^2 = x$ has more than one solution.

116. **Always, sometimes, or never?** A formula can be rearranged to make a different letter the subject.

117. **True or false?** $4x + 1$ has a constant term 1.

118. **Always, sometimes, or never?** $x + y = y + x$ is an identity.
