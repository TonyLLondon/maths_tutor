## Getting started

1. Is $2 + 3 = 5$ true or false?
2. Is $4 + 1 = 6$ true or false?
3. Find the missing number: □ + 2 = 6.
4. Find the missing number: 2 × □ = 6.
5. If $x = 2$, is $x + 1 = 3$ true or false?
6. If $x = 4$, is $x + 1 = 3$ true or false?
7. Simplify $x + x$.
8. Do $x + x$ and $2x$ give the same answer for every $x$?

## Problem (AO3)

## More reasoning (AO2)

93. **True or false?** An identity can be verified by infinitely many tests but one counter-example disproves it.

94. **Always, sometimes, or never?** $x + 5 = 12$ is false when $x = 0$.

96. **True or false?** Substituting into both sides of a suspected identity is a valid check.

98. Nina claims $x^2 + 1 = (x + 1)^2$. Test $x = 1$ and explain.

99. **True or false?** $x = 5$ is a solution to $2x = 10$.

100. **Always, sometimes, or never?** An equation can be true for no values of $x$.

101. **True or false?** $3(x + 2) = 3x + 2$ is an identity.

102. **Always, sometimes, or never?** $x^2 = -1$ has a real solution.

103. Omar says testing $x = 0$ proves $x + x = 2x$. Is that enough? Explain.

104. **True or false?** $x + 2 = 5$ and $5 = x + 2$ mean the same thing.

105. **Always, sometimes, or never?** $(x + 1)^2 \geq 0$ for real $x$.

106. **True or false?** $2x - 1 = x + 3$ has solution $x = 4$.

107. **Always, sometimes, or never?** Identities use $\equiv$ in formal writing but $=$ in many textbooks.
