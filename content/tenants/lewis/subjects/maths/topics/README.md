# Topic worksheets (Lewis household)

One file per GCSE seed topic:

```
content/tenants/lewis/subjects/maths/topics/{domain}/{CODE}.md
```

Examples:

- `topics/number/N4.md` → `/t/lewis/subjects/maths/number/N4`
- `topics/ratio/R1.md` → `/t/lewis/subjects/maths/ratio/R1`

Legacy one-off sheets may stay in `worksheets/` with flat URLs.

Use frontmatter `topic`, `domain`, `title`, optional `week`, `ao`, `printNotes`.
Body sections: Fluency, Reasoning, Problem, Stretch.
