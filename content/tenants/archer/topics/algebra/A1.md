## Fluency (AO1)

1. Find the missing number: □ + 7 = 15.
2. Find $n$ if $n - 4 = 9$.
3. A function machine: **input** → × 3 → + 2 → **output**. What is the output when the input is 5?
4. Work backwards: the output is 14 after × 2 then + 6. What was the input?

## Reasoning (AO2)

5. Maya says □ + 8 = 8 + □ is always true for any number in the box. Is she correct? Explain in one or two sentences.

6. **Always, sometimes, or never?** A function machine with “+ 5” then “− 5” gives the same output as the input.

## Problem (AO3)

7. An Oyster top-up machine adds a £2 bonus to every top-up. After the bonus, Leo’s balance shows £17. How much did he top up before the bonus?

## Stretch

8. Design a two-step function machine that turns 4 into 20. Write the steps and test your machine on input 7.
