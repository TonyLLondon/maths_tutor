## Fluency (AO1)

1. Write the coordinates of a point that is 3 units right and 2 units up from the origin.
2. Complete the table for $y = x + 1$.

   | $x$ | 0 | 1 | 2 | 3 |
   |---|---|---|---|---|
   | $y$ |   |   |   |   |

3. The points $(0, 1)$, $(1, 2)$, $(2, 3)$ lie on a straight line. What is $y$ when $x = 4$?
4. Which point is on the $x$-axis: $(0, 4)$, $(3, 0)$, or $(2, 2)$?

## Reasoning (AO2)

5. A line goes through $(0, 0)$ and $(1, 3)$. Does it also go through $(2, 6)$? Explain how you know without plotting every point.

6. **True or false?** Every point with $y = x + 1$ has $y$ one more than $x$. Give one example to support your answer.

## Problem (AO3)

7. A London taxi charges a fixed £3 hire fee plus £2 per mile. Complete the table and describe what the graph of cost against miles would look like.

   | Miles | 0 | 1 | 2 | 3 |
   |---|---|---|---|---|
   | Cost (£) |   |   |   |   |

## Stretch

8. Write coordinates for three points on a straight line that is neither horizontal nor vertical. Describe in words how $y$ changes when $x$ increases by 1.
