## Getting started

1. Find the missing number: □ + 2 = 5.
2. Find the missing number: 4 + □ = 6.
3. Find the missing number: □ − 1 = 3.
4. Find the missing number: 7 − □ = 5.
5. A machine adds 1. What comes out when 4 goes in?
6. A machine doubles a number. What comes out when 3 goes in?
7. Find the missing number: 2 × □ = 8.
8. Find $n$ if $n + 3 = 7$.

## Fluency (AO1)

9. Find the missing number: □ + 7 = 15.
10. Find the missing number: □ − 5 = 9.
11. Find $n$ if $n - 4 = 9$.
12. Find $n$ if $n + 6 = 20$.
13. A function machine: **input** → × 3 → + 2 → **output**. What is the output when the input is 5?
14. A function machine: **input** → + 4 → × 2 → **output**. What is the output when the input is 3?
15. Work backwards: the output is 14 after × 2 then + 6. What was the input?
16. Work backwards: the output is 20 after + 5 then × 2. What was the input?
17. Find the missing number: 3 × □ = 21.
18. Find the missing number: □ ÷ 4 = 6.
19. Find the missing number: 2 × □ + 1 = 11.
20. Complete the table for the function machine "× 5".

    | Input  | 1 | 2 | 3 | 4 |
    |---|---|---|---|---|
    | Output |   |   |   |   |

## Reasoning (AO2)

21. Maya says □ + 8 = 8 + □ is always true for any number in the box. Is she correct? Explain in one or two sentences.

22. **Always, sometimes, or never?** A function machine with “+ 5” then “− 5” gives the same output as the input.

23. Sam says the missing number in □ × 0 = 0 could be any number. Is Sam correct? Explain your answer.

24. Two function machines both start with input 4. Machine A does “× 2” then “+ 3”. Machine B does “+ 3” then “× 2”. Do they give the same output? **Show both.**

25. Priya says if □ + 5 = 12 then □ must be 7, because 12 − 5 = 7. Explain why her method works, using the words “inverse operation”.

26. **Always, sometimes, or never?** Doubling a number and then halving it gives you back the number you started with.

27. Leo builds a function machine: **input** → ÷ 2 → **output**. He says an input of 10 gives an output of 20. Explain Leo’s mistake and give the correct output.

28. A machine has two unknown steps that turn 3 into 13. One possible pair of steps is “× 4 then + 1”. Find a **different** pair of steps that also works.

## Problem (AO3)

29. An Oyster top-up machine adds a £2 bonus to every top-up. After the bonus, Leo’s balance shows £17. How much did he top up before the bonus?

30. A car park charges a £3 flat fee plus £2 for every hour. Archer pays £13 in total. Use inverse operations to find how many hours he parked.

31. A shop adds a free “bonus pack” of 4 stickers to every order. After the bonus, Sloan has 22 stickers. Using inverse operations, find how many stickers were in the order before the bonus.

32. A theme park ride needs riders to be at least 120 cm tall. Archer’s height with his trainers on is given by □ + 15 = 132, where □ is his height without trainers, in cm. Find Archer’s height without trainers.

33. Think of a number, double it, then add 6. The answer is 20. What was the number?

34. A cinema ticket machine charges £6 per ticket plus a £2 booking fee. The total charged was £26. Use inverse operations to find how many tickets were bought.

## More practice

35. Find the missing number: ___ + 7 = 15.

36. Find the missing number: 5 × ___ = 35.

37. A function machine: input → **×3** → output. If the input is 6, what is the output?

38. Find the missing number: ___ − 9 = 12.

39. A function machine: input → **+4** → **×2** → output. If the input is 5, what is the output?

40. Find the missing number: 3(___ + 2) = 21.

## Stretch

41. Design a two-step function machine that turns 4 into 20. Write the steps and test your machine on input 7.

42. Design a function machine with **three** steps that turns 2 into 19. Show your steps and test it on input 5.

43. Find pairs of **different** whole numbers that go in the two boxes so that □ + □ = 10. How many different pairs can you find?

44. A machine turns input $x$ into an output using the rule “double it, then add the input again”. Show that this always gives the same result as “× 3”. Test your rule with $x = 5$.
