## Fluency (AO1)

1. Find the missing number: □ + 7 = 15.
2. Find the missing number: □ − 5 = 9.
3. Find $n$ if $n - 4 = 9$.
4. Find $n$ if $n + 6 = 20$.
5. A function machine: **input** → × 3 → + 2 → **output**. What is the output when the input is 5?
6. A function machine: **input** → + 4 → × 2 → **output**. What is the output when the input is 3?
7. Work backwards: the output is 14 after × 2 then + 6. What was the input?
8. Work backwards: the output is 20 after + 5 then × 2. What was the input?
9. Find the missing number: 3 × □ = 21.
10. Find the missing number: □ ÷ 4 = 6.
11. Find the missing number: 2 × □ + 1 = 11.
12. Complete the table for the function machine "× 5".

    | Input  | 1 | 2 | 3 | 4 |
    |---|---|---|---|---|
    | Output |   |   |   |   |

## Reasoning (AO2)

13. Maya says □ + 8 = 8 + □ is always true for any number in the box. Is she correct? Explain in one or two sentences.

14. **Always, sometimes, or never?** A function machine with “+ 5” then “− 5” gives the same output as the input.

15. Sam says the missing number in □ × 0 = 0 could be any number. Is Sam correct? Explain your answer.

16. Two function machines both start with input 4. Machine A does “× 2” then “+ 3”. Machine B does “+ 3” then “× 2”. Do they give the same output? **Show both.**

17. Priya says if □ + 5 = 12 then □ must be 7, because 12 − 5 = 7. Explain why her method works, using the words “inverse operation”.

18. **Always, sometimes, or never?** Doubling a number and then halving it gives you back the number you started with.

19. Leo builds a function machine: **input** → ÷ 2 → **output**. He says an input of 10 gives an output of 20. Explain Leo’s mistake and give the correct output.

20. A machine has two unknown steps that turn 3 into 13. One possible pair of steps is “× 4 then + 1”. Find a **different** pair of steps that also works.

## Problem (AO3)

21. An Oyster top-up machine adds a £2 bonus to every top-up. After the bonus, Leo’s balance shows £17. How much did he top up before the bonus?

22. A car park charges a £3 flat fee plus £2 for every hour. Archer pays £13 in total. Use inverse operations to find how many hours he parked.

23. A shop adds a free “bonus pack” of 4 stickers to every order. After the bonus, Sloan has 22 stickers. Using inverse operations, find how many stickers were in the order before the bonus.

24. A theme park ride needs riders to be at least 120 cm tall. Archer’s height with his trainers on is given by □ + 15 = 132, where □ is his height without trainers, in cm. Find Archer’s height without trainers.

25. Think of a number, double it, then add 6. The answer is 20. What was the number?

26. A cinema ticket machine charges £6 per ticket plus a £2 booking fee. The total charged was £26. Use inverse operations to find how many tickets were bought.

## Stretch

27. Design a two-step function machine that turns 4 into 20. Write the steps and test your machine on input 7.

28. Design a function machine with **three** steps that turns 2 into 19. Show your steps and test it on input 5.

29. Find pairs of **different** whole numbers that go in the two boxes so that □ + □ = 10. How many different pairs can you find?

30. A machine turns input $x$ into an output using the rule “double it, then add the input again”. Show that this always gives the same result as “× 3”. Test your rule with $x = 5$.
