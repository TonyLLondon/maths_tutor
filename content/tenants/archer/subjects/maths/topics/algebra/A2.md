## Fluency (AO1)

1. If $x = 3$, find the value of $x + 5$.
2. If $x = 4$, find the value of $2x$.
3. If $x = 5$, find the value of $3x - 2$.
4. If $x = 2$, find the value of $4x + 1$.
5. If $a = 6$ and $b = 2$, find the value of $a + b$.
6. If $a = 6$ and $b = 2$, find the value of $a - b$.
7. If $a = 3$ and $b = 5$, find the value of $ab$.
8. If $x = 4$, find the value of $x^2$.
9. If $x = 3$, find the value of $2x^2$.
10. If $y = 10$, find the value of $\dfrac{y}{2} + 1$.
11. If $x = -2$, find the value of $x + 6$.
12. If $x = -3$, find the value of $5x$.

## Reasoning (AO2)

13. Priya substitutes $x = 2$ into $3x + 1$ and gets 9. What did Priya do wrong? Find the correct value.

14. **Always, sometimes, or never?** Substituting $x = 0$ into $5x + 4$ gives the value 4.

15. Sam says that if $x = -1$, then $x^2 = -1$. Is Sam correct? Explain, showing the correct value.

16. Two expressions, $2x + 3$ and $x + 8$, are both evaluated at $x = 5$. Do they give the same value? **Show your working for both.**

17. **True or false?** For any value of $x$, the expression $x + x$ always gives the same result as $2x$. Test with $x = 7$ to support your answer.

18. Leo says that in the expression $4x^2$, you should multiply $4 \times x$ first, then square the answer. Explain why this is wrong, using $x = 3$ to show the correct method.

19. Compare $x^2$ and $2x$ when $x = 2$. Which is bigger? Now compare them again when $x = 5$. What do you notice?

20. Mo substitutes $x = -4$ into $x^2 + 3$ and gets $-13$. Explain Mo's mistake and find the correct value.

## Problem (AO3)

21. The cost in pounds of a taxi journey is $C = 3 + 2m$, where $m$ is the number of miles. Find $C$ when $m = 5$.

22. Archer's savings after $w$ weeks are given by $S = 20 + 4w$. Find his savings after 6 weeks.

23. The area of a rectangle is given by $A = lw$. Find $A$ when $l = 8$ cm and $w = 5$ cm.

24. A phone plan's monthly cost is $P = 10 + 0.05t$, where $t$ is the number of texts sent. Find the cost for a month with 40 texts.

25. The perimeter of a square with side length $s$ is $P = 4s$. Find the perimeter when $s = 7$ cm, then find the perimeter when the side length doubles to $s = 14$ cm.

26. Sloan's football team scores points using $Pts = 3w + d$, where $w$ is wins and $d$ is draws. Find the points for a season with 6 wins and 4 draws.

## Stretch

27. Find the value of $2x^2 - 3x + 1$ when $x = 4$.

28. Find the value of $\dfrac{a^2 + b}{2}$ when $a = 4$ and $b = 2$.

29. Show that $(x + 2)^2$ and $x^2 + 4$ give **different** values when $x = 3$. Calculate both to prove it.

30. The formula for the $n$th term of a sequence is $3n - 1$. Substitute $n = 1, 2, 3, 4$ to find the first four terms, then predict the 10th term.
