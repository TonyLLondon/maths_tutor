## Getting started

1. Write the missing number: 3 + □ = 5.
2. Write the missing number: □ + 4 = 7.
3. Which letter is used in $x + 2$?
4. How many lots of $x$ are in $2x$?
5. If $x = 3$, what is $x + 1$?
6. If $y = 2$, what is $y + y$?
7. Complete the true number sentence: 6 = □ + 1.
8. Is $2 + 3 = 5$ true or false?

## Fluency (AO1)

9. Write an expression for "a number, $n$, plus 5".
10. Write an expression for "3 times a number, $x$".
11. Write an expression for "a number, $y$, take away 4".
12. In the expression $3x + 2$, what is the coefficient of $x$?
13. In the expression $5x - 7$, what is the constant term?
14. How many terms are in the expression $2x + 3y - 1$?
15. Is $2x + 3 = 9$ an **expression** or an **equation**?
16. Is $3x + 6$ an **expression** or an **equation**?
17. Write an expression for "double a number, $t$, then add 3".
18. Write an expression for the perimeter of a square with side length $s$.
19. Write an equation that says "a number, $x$, plus 7 equals 15".
20. Simplify $x + x + x$ by writing it as a single term.


## More fluency

21. Is $4x + 1$ an expression or an equation?

22. Write an expression for “5 more than a number $n$”.

## Reasoning (AO2)

23. Priya says $2x + 3$ is an equation because it has an $x$ in it. Explain why Priya is wrong.

24. **Always, sometimes, or never?** An expression can be solved to find a single value of $x$.

25. Sam writes the identity $2(x + 3) \equiv 2x + 6$. Explain what the $\equiv$ symbol means and why it is used here instead of $=$.

26. Explain the difference between an **expression**, an **equation**, and a **formula**, using one example of each.

27. **True or false?** $x + x + x = 3x$ is true for every possible value of $x$, so it could be written as an identity.

28. Leo says the perimeter formula $P = 4s$ for a square is an equation. Explain why it is better described as a **formula**.

29. Compare $5x + 2$ and $5x + 2 = 17$. What can you find from the second one that you cannot find from the first?

30. Mo says "an equation always has an equals sign, so $2x + 3$ must be an equation." What is wrong with Mo's reasoning?

## Problem (AO3)

31. Archer is $x$ years old. Sloan is 3 years older. Write an expression for Sloan's age.

32. A rectangle has width $w$ and length twice the width. Write an expression for the length.

33. A taxi charges a £3 flat fee plus £2 per mile, $m$. Write an expression for the total cost.

34. A packet of stickers costs $p$ pence. Write an expression for the cost of 5 packets.

35. Sloan has $n$ marbles. Archer has 4 fewer marbles than twice Sloan's amount. Write an expression for Archer's marbles.

36. The total age of three siblings is 27. The youngest is $x$, the middle one is $x + 2$, and the eldest is $x + 5$. Write and simplify an expression for the total age in terms of $x$.

## Stretch

37. Write an expression for "the square of a number, $x$, plus twice the number".

38. Show that $3(x + 2) + 1$ and $3x + 7$ are equivalent expressions by expanding the first one.

39. A gardener charges a call-out fee of £$c$ plus £$r$ for every hour worked. Write a formula for the total charge, $T$, for $h$ hours of work.

40. Explain, using your own examples, why $2x + 3$ is an expression, $2x + 3 = 9$ is an equation, and $2(x + 3) \equiv 2x + 6$ is an identity.


## Mixed practice

41. Sort these into expression or equation: $2x$, $x + 3 = 10$, $P = 2l + 2w$.

42. Write a formula for the cost $C$ in pounds of $n$ notebooks at £2.50 each.
