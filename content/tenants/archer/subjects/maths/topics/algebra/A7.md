## Fluency (AO1)

1. A function is $f(x) = x + 4$. Find $f(3)$.
2. A function is $f(x) = 2x$. Find $f(5)$.
3. A function is $f(x) = 3x - 1$. Find $f(4)$.
4. A function is $f(x) = x^2$. Find $f(3)$.
5. A mapping diagram sends 1 → 4, 2 → 5, 3 → 6. What rule maps the input to the output?
6. A mapping diagram sends 1 → 2, 2 → 4, 3 → 6. What rule maps the input to the output?
7. If $f(x) = x + 4$ and $f(x) = 10$, what is the input, $x$?
8. If $f(x) = 2x$ and $f(x) = 16$, what is the input, $x$?
9. Complete the table for $f(x) = x + 3$.

   | Input ($x$) | 0 | 1 | 2 | 3 |
   |---|---|---|---|---|
   | Output $f(x)$ |   |   |   |   |

10. Complete the table for $f(x) = 2x - 1$.

    | Input ($x$) | 1 | 2 | 3 | 4 |
    |---|---|---|---|---|
    | Output $f(x)$ |   |   |   |   |

11. A function $f(x) = 5x$ has output 30. What was the input?
12. Which input to $f(x) = x^2$ gives an output of 25?

## Reasoning (AO2)

13. Priya says $f(x) = x + 4$ and $f(3) = 4 + 3$ mean the same thing. Explain what $f(3)$ actually means, and find its value.

14. **Always, sometimes, or never?** For the function $f(x) = x + 1$, the output is greater than the input.

15. Sam says the function $f(x) = x^2$ can never have two different inputs give the same output. Test this using $x = 2$ and $x = -2$, and explain whether Sam is right.

16. A function machine doubles the input, then adds 3. Write this as a function $f(x)$, and find $f(6)$.

17. **True or false?** If $f(x) = 3x$, then $f(a + b) = f(a) + f(b)$ for any numbers $a$ and $b$. Test with $a = 2$ and $b = 3$.

18. Leo says the inverse of $f(x) = x + 5$ is found by adding 5 again. Explain Leo's mistake and describe the correct inverse function.

19. Compare $f(x) = x + 2$ and $g(x) = 2x$. Find $f(3)$ and $g(3)$. Which is bigger? Now try $x = 10$. What changes?

20. Mo says that if $f(2) = 7$ for $f(x) = x + 5$, then $f(3)$ must also be 7 because "functions always give the same output." Explain why Mo is wrong.

## Problem (AO3)

21. A vending machine function takes the number of snacks bought, $n$, and outputs the cost in pence using $f(n) = 80n$. Find the cost of 4 snacks.

22. A function models Archer's savings after $w$ weeks: $f(w) = 15 + 5w$. Find his savings after 8 weeks.

23. A cinema's pricing function is $f(t) = 6t + 2$, where $t$ is the number of tickets, giving the total cost in pounds. If the total cost was £26, find the number of tickets, $t$.

24. A function converts hours, $h$, worked into pay in pounds using $f(h) = 12h$. Sloan earned £84. Find how many hours she worked.

25. A parking function charges $f(m) = 2 + 3m$ pounds for $m$ minutes over a free period. If Leo paid £17, find $m$.

26. A recipe function gives baking time in minutes for a cake of mass $x$ grams: $f(x) = 20 + 0.5x$. Find the baking time for a 400 g cake.

## Stretch

27. A function is $f(x) = x^2 - 1$. Find $f(4)$ and $f(-4)$. What do you notice, and why does this happen?

28. Find the inverse function of $f(x) = x + 7$. Check your answer by applying it to $f(3)$.

29. Find the inverse function of $f(x) = 3x - 2$. Check your answer using $x = 5$.

30. A composite process applies $f(x) = x + 1$ first, then $g(x) = 2x$ to the result. Find the overall output for an input of 4, and write a single function $h(x)$ that describes the whole process.
