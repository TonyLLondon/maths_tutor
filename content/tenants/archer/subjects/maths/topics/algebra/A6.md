## Fluency (AO1)

1. Test $x = 2$ in $2x + 1 = 5$. Is the equation true for this value?
2. Test $x = 3$ in $2x + 1 = 5$. Is the equation true for this value?
3. Expand both sides to check: does $2(x + 3) = 2x + 6$ hold for $x = 1$?
4. Expand both sides to check: does $2(x + 3) = 2x + 6$ hold for $x = 10$?
5. Solve $x + 4 = 9$ to find the one value of $x$ that makes it true.
6. Expand $3(x + 1)$ and compare it with $3x + 1$. Are they the same expression?
7. Test $x = 0$ in $x^2 = x$. Is it true for this value?
8. Test $x = 1$ in $x^2 = x$. Is it true for this value?
9. Test $x = 2$ in $x^2 = x$. Is it true for this value?
10. Expand $5(x - 2)$ and $5x - 10$. Are these always equal?
11. Solve $3x = 12$ to find the value of $x$ that makes it true.
12. Is $x^2 = x$ true for **every** value of $x$, or only **some** values?

## Reasoning (AO2)

13. Priya says $2x + 3 = 7$ is an identity because it's true when $x = 2$. Explain why Priya is wrong.

14. **Always, sometimes, or never?** $3(x + 2) = 3x + 6$ is true.

15. Sam says $x + x = 2x$ is an equation with solution $x = 0$. Explain Sam's mistake, and say what type of statement this really is.

16. Two statements look similar: $2x + 4 = 10$ and $2(x + 2) \equiv 2x + 4$. Test each with $x = 3$ and $x = 5$, and explain what the results tell you about each statement.

17. **True or false?** An identity has infinitely many solutions, but an equation usually has just one.

18. Leo says $x^2 = 4$ is an identity because squaring works for lots of numbers. Explain why Leo is wrong, and find the value(s) of $x$ that actually make it true.

19. Compare $4(x + 1) = 4x + 4$ with $4(x + 1) = 20$. Which one is an identity, and which is an equation? Explain the difference.

20. Mo tests $2x - 1 = x + 3$ with $x = 1$ and finds it false, then with $x = 4$ and finds it true. Explain what this tells you about the type of statement it is.

## Problem (AO3)

21. Sloan says the statement $2(3x) = 6x$ is true for every number she tries. Test it with $x = 4$ and $x = -2$, then say what kind of statement it is.

22. Archer is checking a shopkeeper's till receipt formula: "total = price × quantity + delivery". He tests it with two different orders and it works both times. Explain why testing with numbers alone cannot **prove** a formula is always true.

23. A game rule says "double the score, then subtract the original score" always leaves you with the score you started with. Write this as an identity using $s$ for score, and check it with $s = 6$.

24. A student claims $x + 5 = 5 + x$ is an equation you must solve to find $x$. Explain why this claim is incorrect, and identify what kind of statement it really is.

25. Test whether $(x + 1)^2 = x^2 + 1$ is an identity by trying $x = 2$. What does your result show?

26. A market trader says "half the price, doubled, always gives back the original price" for any price $p$. Write this as an identity and verify it with $p = 14$.

## Stretch

27. Show that $3(x + 4) - 2x \equiv x + 12$ by expanding and simplifying the left-hand side.

28. Show that $2(x - 1) + 2(x + 1) \equiv 4x$ by expanding both brackets and simplifying.

29. Decide whether $(x + 2)^2 \equiv x^2 + 4$ is an identity. Test with $x = 1$ and $x = 3$ to support your conclusion.

30. Create your own pair of expressions that form a **true identity**, and a different pair that only forms an equation with one solution. Prove each with substitution.
