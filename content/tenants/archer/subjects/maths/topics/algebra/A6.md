## Getting started

1. Is $2 + 3 = 5$ true or false?
2. Is $4 + 1 = 6$ true or false?
3. Find the missing number: □ + 2 = 6.
4. Find the missing number: 2 × □ = 6.
5. If $x = 2$, is $x + 1 = 3$ true or false?
6. If $x = 4$, is $x + 1 = 3$ true or false?
7. Simplify $x + x$.
8. Do $x + x$ and $2x$ give the same answer for every $x$?

## Fluency (AO1)

9. Test $x = 2$ in $2x + 1 = 5$. Is the equation true for this value?
10. Test $x = 3$ in $2x + 1 = 5$. Is the equation true for this value?
11. Expand both sides to check: does $2(x + 3) = 2x + 6$ hold for $x = 1$?
12. Expand both sides to check: does $2(x + 3) = 2x + 6$ hold for $x = 10$?
13. Solve $x + 4 = 9$ to find the one value of $x$ that makes it true.
14. Expand $3(x + 1)$ and compare it with $3x + 1$. Are they the same expression?
15. Test $x = 0$ in $x^2 = x$. Is it true for this value?
16. Test $x = 1$ in $x^2 = x$. Is it true for this value?
17. Test $x = 2$ in $x^2 = x$. Is it true for this value?
18. Expand $5(x - 2)$ and $5x - 10$. Are these always equal?
19. Solve $3x = 12$ to find the value of $x$ that makes it true.
20. Is $x^2 = x$ true for **every** value of $x$, or only **some** values?


## More fluency

21. Is $2x + 6 = 2(x + 3)$ an identity or an equation?

22. Show $x + x = 2x$ is true when $x = 4$.

## Reasoning (AO2)

23. Priya says $2x + 3 = 7$ is an identity because it's true when $x = 2$. Explain why Priya is wrong.

24. **Always, sometimes, or never?** $3(x + 2) = 3x + 6$ is true.

25. Sam says $x + x = 2x$ is an equation with solution $x = 0$. Explain Sam's mistake, and say what type of statement this really is.

26. Two statements look similar: $2x + 4 = 10$ and $2(x + 2) \equiv 2x + 4$. Test each with $x = 3$ and $x = 5$, and explain what the results tell you about each statement.

27. **True or false?** An identity has infinitely many solutions, but an equation usually has just one.

28. Leo says $x^2 = 4$ is an identity because squaring works for lots of numbers. Explain why Leo is wrong, and find the value(s) of $x$ that actually make it true.

29. Compare $4(x + 1) = 4x + 4$ with $4(x + 1) = 20$. Which one is an identity, and which is an equation? Explain the difference.

30. Mo tests $2x - 1 = x + 3$ with $x = 1$ and finds it false, then with $x = 4$ and finds it true. Explain what this tells you about the type of statement it is.

## Problem (AO3)

31. Sloan says the statement $2(3x) = 6x$ is true for every number she tries. Test it with $x = 4$ and $x = -2$, then say what kind of statement it is.

32. Archer is checking a shopkeeper's till receipt formula: "total = price × quantity + delivery". He tests it with two different orders and it works both times. Explain why testing with numbers alone cannot **prove** a formula is always true.

33. A game rule says "double the score, then subtract the original score" always leaves you with the score you started with. Write this as an identity using $s$ for score, and check it with $s = 6$.

34. A student claims $x + 5 = 5 + x$ is an equation you must solve to find $x$. Explain why this claim is incorrect, and identify what kind of statement it really is.

35. Test whether $(x + 1)^2 = x^2 + 1$ is an identity by trying $x = 2$. What does your result show?

36. A market trader says "half the price, doubled, always gives back the original price" for any price $p$. Write this as an identity and verify it with $p = 14$.

## Stretch

37. Show that $3(x + 4) - 2x \equiv x + 12$ by expanding and simplifying the left-hand side.

38. Show that $2(x - 1) + 2(x + 1) \equiv 4x$ by expanding both brackets and simplifying.

39. Decide whether $(x + 2)^2 \equiv x^2 + 4$ is an identity. Test with $x = 1$ and $x = 3$ to support your conclusion.

40. Create your own pair of expressions that form a **true identity**, and a different pair that only forms an equation with one solution. Prove each with substitution.


## Mixed practice

41. Explain why $2x + 3 = 10$ is an equation but $2x + 3$ is not.

42. Is $(x + 1)^2 = x^2 + 2x + 1$ an identity? Test with $x = 3$.
