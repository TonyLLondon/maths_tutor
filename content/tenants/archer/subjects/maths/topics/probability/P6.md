## Getting started

1. List the two possible outcomes of flipping one coin.
2. Choose one top: red or blue. Choose one hat: green or yellow. How many outfits can you make?
3. List the numbers you can make using one digit from 1, 2, and 3.
4. Put A and B in every possible order.
5. A snack can be an apple or a banana, and a drink can be water or milk. How many snack-and-drink choices are there?
6. List every pair made by choosing one of A or B and one of 1 or 2.
7. You have 3 shirts and 1 pair of trousers. How many different outfits can you make?
8. Why is it helpful to write a list in a clear order?
9. List both orders of the letters A and B.
10. Soup is tomato or vegetable, and bread is white or brown. How many different meals are there?
11. A code uses digit 1 or 2 and letter A or B. How many different codes are there?
12. There are 2 games and 3 snacks. How many game-and-snack choices are there?

## Fluency (AO1)

13. List all two-digit numbers that can be made using the digits 1, 2, and 3, without repeating a digit.
14. How many different ways can you arrange the letters A, B, and C in a row?
15. List all the possible outcomes when choosing one starter (soup, salad) and one main (pasta, fish, chicken) from a menu.
16. A padlock code uses two digits chosen from 1, 2, 3, 4 without repeats. List all possible codes.
17. List all the two-digit numbers **greater than 40** that can be made from the digits 3, 4, and 5 (digits may repeat).
18. How many outfits can be made by choosing one of 3 shirts and one of 2 pairs of trousers? List them systematically.
19. List all the possible ways to arrange the letters X, Y, Z in a row.
20. A coin is flipped and a dice is rolled. List all the possible outcomes systematically (e.g. H1, H2, ...).
21. List all three-digit numbers that can be made from the digits 1 and 2 (digits may repeat).
22. A sandwich shop offers 2 breads and 4 fillings. List all possible sandwiches systematically, and state how many there are.
23. List all the ways of choosing 2 different letters from A, B, C, D (order does not matter).
24. How many different two-digit numbers can be formed using the digits 5, 6, 7, and 8 without repeating a digit? List them systematically, starting with the smallest.

## Reasoning (AO2)

25. Archer lists the outcomes for two coin flips as HH, HT, TT and says there are 3 outcomes. Explain what Archer has missed, and correct the list.

26. Explain why listing outcomes in a systematic order (for example, starting each group with the same first item) helps avoid missing or repeating outcomes.

27. Sloan says choosing 2 letters from {A, B, C} gives 6 possible pairs. Is Sloan correct if order does not matter? List the pairs to check.

28. **True or false?** When listing outcomes for two independent events, you can multiply the number of options for each event to find the total number of outcomes. Give an example.

29. A student lists two-digit numbers from digits 1, 2, 3 (no repeats) as 12, 13, 21, 23, 31. Explain what is missing from this list.

30. Explain the difference between listing arrangements where **order matters** (e.g. arranging letters) and where **order does not matter** (e.g. choosing a pair), using an example of each.

31. Priya says that for 3 shirts and 4 pairs of trousers, there are 3 + 4 = 7 possible outfits. Is Priya correct? Explain the correct method and answer.

## Problem (AO3)

32. A café offers 3 sandwiches, 2 drinks, and 2 desserts as part of a set meal. Describe a systematic way to count the total number of possible meals, and find how many there are.

33. A four-digit PIN uses only the digits 1 and 2 (digits may repeat). How many different PINs are possible? Explain your systematic method.

34. Three friends — Archer, Sloan, and Priya — are lining up for a photo. List all the possible orders they could stand in, and state how many there are.

35. A code for a safe uses 2 different letters chosen from the set {A, B, C, D, E}, where order matters (so AB is different from BA). How many codes are possible? Explain your reasoning.

36. A restaurant menu has 5 starters and 6 mains. A customer chooses one of each. How many different combinations are possible? Show a systematic method (not just multiplying) for a smaller version of this problem to check your answer.

37. List all the ways to make 50p using only 20p and 10p coins.

38. A school has 4 house colours (red, blue, green, yellow) and wants to pick 2 different colours for a new kit (order does not matter). List all the possible pairs, and state how many there are.

## Stretch

39. Explain a systematic strategy you could use to list outcomes of rolling three dice and finding which show "doubles or triples" (at least two dice showing the same number), without listing all 216 outcomes individually.

40. A menu has *s* starters, *m* mains, and *d* desserts. Write a general rule (formula) for the total number of possible three-course meals, and explain why this rule works using the idea of systematic listing.

41. Explain why systematically listing outcomes is important when calculating probabilities, using an example where a non-systematic list might miss an outcome.

42. Investigate: how many different ways can 4 people be arranged in a queue? Describe your systematic method and give the total.
