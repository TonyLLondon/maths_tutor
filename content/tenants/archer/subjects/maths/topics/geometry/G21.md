## Fluency (AO1)

1. Write the exact value of $\sin 30°$.
2. Write the exact value of $\cos 30°$.
3. Write the exact value of $\tan 45°$.
4. Write the exact value of $\sin 45°$.
5. Write the exact value of $\cos 45°$.
6. Write the exact value of $\sin 60°$.
7. Write the exact value of $\cos 60°$.
8. Write the exact value of $\tan 60°$.
9. Write the exact value of $\sin 0°$.
10. Write the exact value of $\cos 0°$.
11. Write the exact value of $\tan 30°$.
12. Write the exact value of $\sin 90°$.

## Reasoning (AO2)

13. Mia notices that $\sin 30° = \cos 60°$. Explain why these two values are equal, using the idea of complementary angles (angles that add to 90°).

14. **True or false?** $\sin 45° = \cos 45°$.

15. Ravi says $\tan 45° = 1$ means the opposite and adjacent sides of a right-angled triangle with a 45° angle must be equal. Explain why he is correct.

16. **Always, sometimes, or never?** As the angle increases from 0° to 90°, the value of $\sin\theta$ increases.

17. Priya says $\cos 90°$ should be a very small positive number, not exactly 0. Explain why $\cos 90° = 0$ exactly.

18. Explain why $\tan\theta$ is undefined at $\theta = 90°$, using the definition $\tan\theta = \dfrac{\sin\theta}{\cos\theta}$.

19. Sloan says that because $30° + 60° = 90°$, it must also be true that $\sin 30° + \sin 60° = \sin 90°$. Show, using exact values, that this is false.

20. Explain why $\sin\theta$ and $\cos\theta$ swap values between 30° and 60° (i.e. $\sin 30° = \cos 60°$ and $\cos 30° = \sin 60°$).

## Problem (AO3)

21. A right-angled triangle has a hypotenuse of 8 cm and an angle of 30°. Use $\sin 30° = \dfrac{1}{2}$ to find the exact length of the side opposite the 30° angle.

22. A right-angled triangle has a hypotenuse of 10 cm and an angle of 60°. Use $\cos 60° = \dfrac{1}{2}$ to find the exact length of the side adjacent to the 60° angle.

23. A right-angled triangle has one leg of 6 cm and a 45° angle between the hypotenuse and this leg. Use $\tan 45° = 1$ to find the length of the other leg.

24. Use the exact value of $\sin 60°$ to find the exact height of an equilateral triangle with side length 4 cm (split it into two right-angled triangles first).

25. A ramp makes an angle of 30° with the ground and has a horizontal base of 12 m. Use $\tan 30° = \dfrac{1}{\sqrt{3}}$ to find the exact height of the ramp, leaving your answer with a surface root if needed.

26. Show, using exact values, that $\sin^2 45° + \cos^2 45° = 1$ (calculate each part separately, then add).

## Stretch

27. Use exact values to show that $(\sin 30° + \cos 60°) = 1$.

28. A right-angled triangle has angle 60° and the side opposite this angle is $6\sqrt{3}$ cm. Use $\sin 60° = \dfrac{\sqrt{3}}{2}$ to find the exact length of the hypotenuse.

29. Use exact trig values to show that $\tan 60° \div \tan 30° = 3$.

30. Explain how the exact values for 30°, 45°, and 60° can be found using two special right-angled triangles: an isosceles right-angled triangle, and an equilateral triangle cut in half. Describe the side lengths used for one of these triangles.
