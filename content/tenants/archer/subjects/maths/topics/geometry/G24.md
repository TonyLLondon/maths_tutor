## Getting started

1. Start on a square and move 2 squares right. How many horizontal squares did you move?
2. Start on a square and move 3 squares up. How many vertical squares did you move?
3. Which direction is opposite to right?
4. Which direction is opposite to up?
5. Start at (0, 0) and move 2 right. Where do you finish?
6. Start at (0, 0) and move 3 up. Where do you finish?
7. Start at (1, 1) and move 2 right. Where do you finish?
8. A shape moves 4 left and does not move up or down. What is its vertical movement?

## Fluency (AO1)

9. Write the column vector that describes a movement of 3 right and 2 up.
10. Write the column vector that describes a movement of 4 left and 1 up.
11. Write the column vector that describes a movement of 5 right and 3 down.
12. Point A is at $(2, 3)$. It is translated by the vector $\binom{4}{1}$. Find the coordinates of the image point.
13. Point B is at $(0, 0)$. It is translated by the vector $\binom{-3}{5}$. Find the coordinates of the image point.
14. Point C is at $(6, -2)$. It is translated by the vector $\binom{-2}{-4}$. Find the coordinates of the image point.
15. Point D is at $(-1, -1)$. It is translated by the vector $\binom{5}{2}$. Find the coordinates of the image point.
16. Point A is at $(1, 2)$ and its image after translation is $(5, 6)$. Find the column vector of the translation.
17. Point B is at $(3, 7)$ and its image after translation is $(3, 2)$. Find the column vector of the translation.
18. Point C is at $(-4, 2)$ and its image after translation is $(0, 0)$. Find the column vector of the translation.
19. What single word describes a transformation that slides a shape without turning or flipping it?
20. If a shape is translated by $\binom{0}{0}$, what happens to the shape?


## More fluency

21. Write the vector for a translation 3 right and 2 up.

22. Write the vector for 4 left and 1 down.

## Reasoning (AO2)

23. Mia says that translating a shape by $\binom{2}{3}$ makes it move 2 up and 3 right. Explain her mistake.

24. **True or false?** Translation always produces an image that is congruent to the original shape.

25. Ravi translates point $(4, 5)$ by $\binom{-6}{2}$ and gets $(2, 7)$. Check his working and correct it if needed.

26. **Always, sometimes, or never?** Translating a shape changes its size or orientation.

27. Priya says the vector $\binom{3}{-3}$ describes the same movement as $\binom{-3}{3}$. Explain why she is wrong.

28. Explain how you would find the column vector describing the translation from point $A(2, 1)$ to point $B(7, 4)$.

29. Sloan translates a shape by $\binom{5}{0}$. Describe what happens to the shape's position, and explain why the $y$-coordinates of every point stay the same.

30. Leo says that translating a shape twice by the same vector $\binom{2}{1}$ is the same as translating it once by $\binom{4}{2}$. Is he correct? Explain.

## Problem (AO3)

31. Triangle A has vertices $(1, 1)$, $(3, 1)$, and $(1, 4)$. It is translated by the vector $\binom{2}{-3}$. Find the coordinates of the translated triangle's vertices.

32. A shape is translated by $\binom{4}{2}$ and then by $\binom{-1}{3}$. Find the single column vector that describes the overall (combined) translation.

33. A point starts at $(5, 5)$, is translated by $\binom{-2}{4}$, and then translated again by $\binom{3}{-6}$. Find the final coordinates of the point.

34. A robot on a grid starts at $(0, 0)$. It follows three translations in order: $\binom{3}{2}$, $\binom{-1}{4}$, $\binom{2}{-5}$. Find the robot's final position.

35. A quadrilateral has vertices $(0, 0)$, $(2, 0)$, $(2, 3)$, and $(0, 3)$. Find the coordinates after translation by $\binom{-3}{-1}$, then describe whether the shape's size and angles have changed.

36. The vector from point $P$ to point $Q$ is $\binom{6}{-2}$. If $P$ is at $(-1, 5)$, find the coordinates of $Q$.

## Stretch

37. Point $A(3, -2)$ is translated to point $B(-1, 6)$. Find the column vector of this translation, then find the vector that would translate $B$ back to $A$.

38. A shape is translated by vector $\binom{a}{b}$ so that the point $(2, -3)$ maps to $(6, 1)$. Find the values of $a$ and $b$.

39. A shape undergoes two translations, $\binom{x}{2}$ then $\binom{3}{y}$, and the overall effect is the same as a single translation of $\binom{7}{-1}$. Find the values of $x$ and $y$.

40. Design your own translation puzzle: choose a starting point and a column vector, and find the image point. Then find the vector that would translate the image back to the starting point.


## Mixed practice

41. Point (2, 5) is translated by $\begin{pmatrix} 3 \\ -1 \end{pmatrix}$. Find the image.

42. A shape moves 5 units left and 3 units up. Write the translation vector.
