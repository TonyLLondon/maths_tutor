## Fluency (AO1)

1. y is inversely proportional to x, so y = k/x. When x = 3, y = 8. Find k.
2. Using y = 24/x, find y when x = 6.
3. Using y = 24/x, find x when y = 2.
4. Write the equation y = k/x if k = 15.
5. If xy = 20, find y when x = 4.
6. If xy = 20, find x when y = 10.
7. y = k/x and, when x = 5, y = 6. Find k.
8. Using y = 30/x, find y when x = 2.
9. p = k/q. When q = 9, p = 4. Find k.
10. Using p = 36/q, find p when q = 12.
11. Rearrange y = k/x to make x the subject.

## Reasoning (AO2)

12. Mia says that in y = k/x, you can always find k by multiplying x and y together. Is she correct? Explain.

13. **Always, sometimes, or never?** The graph of y = k/x (for positive k and x > 0) gets closer to the x-axis as x increases, but never touches it.

14. **True or false?** You can find k in an inverse proportion equation using just one pair of matching x and y values. Explain.

15. Priya says the equations y = k/x and y = kx describe the same type of relationship, just written differently. Is she correct?

16. Explain what happens to y in the equation y = k/x as x gets very large.

17. Sam rearranges xy = 18 to get y = 18/x. Sloan rearranges it to get x = 18/y. Are both correct?

18. Leo has two equations: y = 12/x and y = x/12. Which one is inverse proportion, and which is direct proportion? Explain how you can tell.

## Problem (AO3)

19. y is inversely proportional to x. When x = 4, y = 15. Find the equation linking x and y, then find y when x = 10.

20. p is inversely proportional to q. When q = 8, p = 5. Find the equation, then find q when p = 20.

21. m is inversely proportional to n. When n = 3, m = 28. Find m when n = 7.

22. t is inversely proportional to s. When s = 2, t = 45. Find the equation, then find t when s = 9.

23. Rearrange the equation y = k/x to make k the subject, then use it to find k when x = 7 and y = 3.

24. Two variables satisfy xy = 100. Complete the missing values: x = 4, y = ?; x = 25, y = ?; x = 50, y = ?.

25. w is inversely proportional to v, with equation w = 72/v. Find v when w = 9.

## Stretch

26. y is inversely proportional to x. When x = 6, y = 10. Find the equation, then find x when y = 4.

27. Two quantities x and y satisfy xy = 144 and x = y. Find x and y.

28. m is inversely proportional to n. When n = 5, m = 8. If n is increased to 20, by what factor does m decrease?

29. Explain why, for inverse proportion y = k/x, x cannot equal 0. What happens to y as x gets closer and closer to 0 (for positive k)?

30. Design your own inverse proportion equation. State the equation, then use it to find both a missing y and a missing x.
