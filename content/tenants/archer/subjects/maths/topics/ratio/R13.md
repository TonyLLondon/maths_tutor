## Getting started

1. Using y = 12/x, find y when x = 2.
2. Using y = 12/x, find y when x = 3.
3. If x × y = 10 and x = 2, find y.
4. If x × y = 12 and y = 3, find x.
5. In y = k/x, which letter is the constant?
6. Complete the inverse proportion equation: y = k ÷ ___.
7. When x = 4 and y = 2, find x × y.
8. In an inverse proportion, if x doubles, does y double or halve?
9. Using y = 18/x, find y when x = 6.
10. If x × y = 24 and x = 8, find y.
11. In inverse proportion, if x is multiplied by 3, y is multiplied by ___.
12. Using y = 30/x, find x when y = 5.

## Fluency (AO1)

13. y is inversely proportional to x, so y = k/x. When x = 3, y = 8. Find k.
14. Using y = 24/x, find y when x = 6.
15. Using y = 24/x, find x when y = 2.
16. Write the equation y = k/x if k = 15.
17. If xy = 20, find y when x = 4.
18. If xy = 20, find x when y = 10.
19. y = k/x and, when x = 5, y = 6. Find k.
20. Using y = 30/x, find y when x = 2.
21. p = k/q. When q = 9, p = 4. Find k.
22. Using p = 36/q, find p when q = 12.
23. Rearrange y = k/x to make x the subject.

## Reasoning (AO2)

24. Mia says that in y = k/x, you can always find k by multiplying x and y together. Is she correct? Explain.

25. **Always, sometimes, or never?** The graph of y = k/x (for positive k and x > 0) gets closer to the x-axis as x increases, but never touches it.

26. **True or false?** You can find k in an inverse proportion equation using just one pair of matching x and y values. Explain.

27. Priya says the equations y = k/x and y = kx describe the same type of relationship, just written differently. Is she correct?

28. Explain what happens to y in the equation y = k/x as x gets very large.

29. Sam rearranges xy = 18 to get y = 18/x. Sloan rearranges it to get x = 18/y. Are both correct?

30. Leo has two equations: y = 12/x and y = x/12. Which one is inverse proportion, and which is direct proportion? Explain how you can tell.

## Problem (AO3)

31. y is inversely proportional to x. When x = 4, y = 15. Find the equation linking x and y, then find y when x = 10.

32. p is inversely proportional to q. When q = 8, p = 5. Find the equation, then find q when p = 20.

33. m is inversely proportional to n. When n = 3, m = 28. Find m when n = 7.

34. t is inversely proportional to s. When s = 2, t = 45. Find the equation, then find t when s = 9.

35. Rearrange the equation y = k/x to make k the subject, then use it to find k when x = 7 and y = 3.

36. Two variables satisfy xy = 100. Complete the missing values: x = 4, y = ?; x = 25, y = ?; x = 50, y = ?.

37. w is inversely proportional to v, with equation w = 72/v. Find v when w = 9.

## Stretch

38. y is inversely proportional to x. When x = 6, y = 10. Find the equation, then find x when y = 4.

39. Two quantities x and y satisfy xy = 144 and x = y. Find x and y.

40. m is inversely proportional to n. When n = 5, m = 8. If n is increased to 20, by what factor does m decrease?

41. Explain why, for inverse proportion y = k/x, x cannot equal 0. What happens to y as x gets closer and closer to 0 (for positive k)?

42. Design your own inverse proportion equation. State the equation, then use it to find both a missing y and a missing x.
