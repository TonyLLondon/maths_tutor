## Getting started

1. Complete: if x = 1 and y = 2, then x : y = ___.
2. If each bag holds 3 apples, how many apples are in 2 bags?
3. Complete the rule: multiply x by 2 to get y, so y = ___x.
4. Using y = 2x, find y when x = 3.
5. Using y = 4x, find y when x = 2.
6. In the rule y = 5x, what number multiplies x?
7. A table has x = 1, 2, 3 and y = 3, 6, 9. Complete the rule y = ___x.
8. Does the rule y = 2x give y = 0 when x = 0?
9. Using y = 3x, find y when x = 4.
10. Using y = 6x, find x when y = 18.
11. If x = 2 and y = 10, complete the rule y = ___ x.
12. A table has x = 0, 1, 2 and y = 0, 4, 8. Complete the rule y = ___ x.

## Fluency (AO1)

13. A table shows x = 2, y = 8 and x = 3, y = 12. What is the ratio x : y in simplest form?
14. Using the ratio x : y = 1 : 4, complete: if x = 5, y = ?
15. A function rule multiplies the input by a constant, k. If an input of 3 gives an output of 15, find k.
16. Write the function rule, in the form y = kx, for a ratio table where x : y = 1 : 6.
17. £1 = $1.25. Write the ratio pounds : dollars, and the rule y = kx for dollars (y) from pounds (x).
18. A conversion table shows 5 miles = 8 km. Find the ratio miles : km in simplest form, and the constant k so that km = k × miles.
19. If y = 3x, find y when x = 7.
20. If y = 0.5x, find x when y = 9.
21. A table shows: x = 1, 2, 3, 4 and y = 4, 8, 12, 16. What is the rule linking x and y?
22. In a table, x : y stays constant at 2 : 7. If x = 6, find y.
23. A recipe function converts number of people (x) to grams of pasta (y) using y = 75x. Find y when x = 4.

## Reasoning (AO2)

24. Mia says every ratio table represents a straight-line graph through the origin. Is she correct? Explain why.

25. **Always, sometimes, or never?** A function of the form y = kx passes through the point (0, 0).

26. **True or false?** If x : y is always 1 : 3, then y is directly proportional to x. Explain.

27. Priya has a table where x : y is 2 : 5 for some rows but 2 : 6 for others. Can this data be described by a single rule y = kx? Explain.

28. Explain how you can find k in y = kx just from one pair of values (x, y), where x ≠ 0.

29. Sam says a conversion graph for pounds to euros must be a straight line through the origin, since £0 = €0. Is Sam correct?

30. Leo says that if you double x in y = kx, then y also doubles. Is Leo correct? Explain using the ratio.

## Problem (AO3)

31. £1 = €1.15. Write the function rule for converting pounds (x) to euros (y), then find how many euros £40 is worth.

32. A recipe uses the ratio people : eggs = 2 : 3. Write the rule y = kx for eggs (y) needed for x people, then find how many eggs are needed for 10 people.

33. A car uses fuel in the ratio distance : fuel = 8 : 1 (km per litre). Write the rule for fuel used, y, from distance travelled, x, then find the fuel used for a 200 km trip.

34. A conversion table shows 4 inches = 10 cm. Find the rule cm (y) = k × inches (x), then convert 18 inches to cm.

35. Archer earns pocket money in the ratio hours worked : pounds earned = 2 : 5. Write the rule and find how much he earns for 6 hours.

36. A taxi fare (excluding any fixed charge) is directly proportional to distance: 3 miles costs £6. Find the rule and the fare for a 10-mile ride.

37. A scale drawing rule converts real metres (x) to drawing centimetres (y), where 1 m of real length becomes 5 cm on the drawing. Find the drawing length for a real wall of 8.4 m.

## Stretch

38. Two variables always have the ratio x : y = 3 : 7. If y = 42, find x, and write the rule linking x and y.

39. A rule y = kx passes through the point (4, 10). Find k, and use it to find y when x = 15.

40. Two conversion rules are given: y = 2.5x (rule A) and y = (5/2)x (rule B). Are these the same rule? Explain.

41. A graph shows a straight line through the origin passing through (6, 15). Explain how to find the equation of the line using the ratio of the coordinates.

42. Design your own real-life ratio table (e.g. cost vs quantity), find the rule y = kx, and use it to predict a new value.
