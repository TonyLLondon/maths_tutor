## Fluency (AO1)

1. $a = 12$ and $b = 5$, both correct to the nearest whole number. Write the error interval for $a$.
2. Using the values in Q1, write the error interval for $b$.
3. Find the maximum possible value of $a + b$ using the bounds above.
4. Find the minimum possible value of $a + b$ using the bounds above.
5. Find the maximum possible value of $a - b$ using the bounds above.
6. Find the minimum possible value of $a - b$ using the bounds above.
7. $x = 20$ cm and $y = 4$ cm, both correct to the nearest whole number. Find the maximum possible value of $x \times y$.
8. Find the minimum possible value of $x \times y$ using the values in Q7.
9. Find the maximum possible value of $x \div y$ using the values in Q7.
10. Find the minimum possible value of $x \div y$ using the values in Q7.

## More fluency

11. $p = 8.0$ and $q = 3.0$, both correct to 1 decimal place. Find the upper bound of $p + q$.
12. Find the lower bound of $p - q$ using the values in Q11.
13. $m = 50$ (nearest 10) and $n = 6$ (nearest whole number). Find the upper bound of $m \div n$.
14. Find the lower bound of $m \div n$ using the values in Q13.

## Reasoning (AO2)

15. Archer says the maximum of $a - b$ is found using the maximum of $a$ and the maximum of $b$. Explain why this is wrong, and state the correct rule.

16. **Always, sometimes, or never?** To find the maximum of $a \div b$, you use the upper bound of $a$ and the lower bound of $b$.

17. Sloan says the minimum of $a + b$ is found using the lower bound of $a$ and the upper bound of $b$. Is she correct? Explain the correct method.

18. **Always, sometimes, or never?** The maximum possible value of a product $a \times b$ uses the upper bounds of both $a$ and $b$.

19. Compare finding the minimum of $a \div b$ with finding the minimum of $a \times b$. Which bounds do you use for each?

## Problem (AO3)

20. A rectangle has length 15 cm and width 8 cm, both correct to the nearest cm. Work out the maximum possible area.

21. Using the same rectangle as Q20, work out the minimum possible area.

22. A car travels 100 miles, to the nearest mile, in 2 hours, to the nearest whole hour. Work out the maximum possible average speed.

23. Two masses are 25 kg and 10 kg, both correct to the nearest kg. Work out the minimum possible value of the difference between them.

24. A stick of length 30 cm (nearest cm) is cut from a length of ribbon 200 cm (nearest 10 cm). Work out the maximum possible length of ribbon left.

## Stretch

25. A cuboid has dimensions 4 cm, 5 cm and 10 cm, all measured to the nearest cm. Work out the maximum possible volume.

26. Using the same cuboid as Q25, work out the minimum possible volume.

27. A speed is calculated from a distance of 90 m (nearest 10 m) and a time of 12 s (nearest whole second). Find the maximum possible speed.

28. Using the same values as Q27, find the minimum possible speed, giving your answer to 2 decimal places.

## Mixed practice

29. Two lengths are 18 cm and 7 cm, both to the nearest cm. Work out the error interval for the sum of the two lengths.

30. Sloan says the minimum possible value of $a \times b$ always uses the lower bounds of $a$ and $b$. Is she correct? Explain.
