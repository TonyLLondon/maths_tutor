## Getting started

1. A toy comes in red or blue. List the two possible colours.
2. You can choose a cat or dog and a small or large bed. How many different choices are there?
3. List all the two-digit numbers you can make using 1 and 2 once each.
4. A coin can land heads or tails. How many possible results are there?
5. List all whole numbers greater than 2 and less than 6.
6. Sam can pick apple, pear or banana. How many choices does Sam have?
7. A spinner has A and B. It is spun twice. Which list is complete: AA, AB, BA, BB or AA, BB?
8. How many different pairs can you make by choosing one shirt from 2 shirts and one hat from 2 hats?
9. A drink is water or juice. How many drink choices are there?
10. List both results when a coin is flipped once: heads and ___.
11. A badge is round or square. How many shapes are there to choose from?
12. A code uses the digit 1 or 2. List both possible digits.

## Fluency (AO1)

13. List all the two-digit numbers you can make using the digits 1, 2 and 3 (using each digit once per number).
14. Archer has a red shirt or a blue shirt, and black shorts or white shorts. List all the possible outfits.
15. List all the factors of 24.
16. How many different ways can you arrange the letters A, B, C in a row? List them.
17. List all the two-digit even numbers you can make from the digits 4, 5 and 6, without repeating digits.
18. A café sells tea or coffee, with milk or no milk. List all the possible drink orders.
19. List all the factors of 36.
20. Three friends — Archer, Sloan and Priya — are in a race. List all the possible orders they could finish in.
21. List all the multiples of 6 between 1 and 40.
22. A coin is flipped and a dice is rolled. List all the possible outcomes (e.g. H1, H2, …).

## More fluency

23. List all the two-digit numbers that can be made from the digits 2, 7 and 9 (each digit used once per number).
24. A sandwich shop offers ham, cheese or tuna fillings, with white or brown bread. List all the possible sandwiches.
25. List all the factors of 40.
26. How many three-digit numbers can be made using only the digits 1 and 2, if digits can repeat? List them.

## Reasoning (AO2)

27. Sloan lists outfits by only counting shirts, forgetting to multiply by the number of shorts. Explain why this gives the wrong total, and describe the correct method.

28. **Always, sometimes, or never?** If there are 3 choices for a starter and 4 choices for a main course, there are 12 possible meals.

29. Archer says listing the outcomes of flipping 2 coins gives 3 outcomes: 0 heads, 1 head, 2 heads. Explain why this is wrong and give the correct list.

30. Explain a systematic way to list all the two-digit numbers with digit sum 9, without missing any.

31. **Always, sometimes, or never?** The number of ways to arrange n different items in a row is n × (n − 1) × (n − 2) × … × 1.

## Problem (AO3)

32. A pizza shop offers 3 bases and 5 toppings, and each pizza has exactly one base and one topping. How many different pizzas are possible?

33. A padlock code uses 2 different digits from 1, 2, 3, 4 (order matters, no repeats). List all the possible codes and say how many there are.

34. Archer, Sloan and two friends want to sit in a row of 4 chairs. In how many different orders can they sit?

35. A café menu has 4 sandwich fillings and 3 types of bread, but tuna cannot go on brown bread. How many different sandwiches are possible?

36. How many different 3-digit codes can be made using the digits 0–9, if repeats are allowed and the first digit cannot be 0?

## Stretch

37. In a class of 30, each pupil shakes hands with every other pupil exactly once. How many handshakes take place?

38. A restaurant has a set menu with 4 starters, 5 mains and 2 desserts. How many different 3-course meals are possible if you must choose one of each?

39. How many ways can 5 books be arranged on a shelf?

40. A 4-character password uses only the letters A–E, and letters can repeat. How many different passwords are possible?

## Mixed practice

41. List all the ways to make 50p using only 20p and 10p coins.

42. **Always, sometimes, or never?** Listing outcomes in a table (sample space diagram) makes it easier to avoid missing any combinations.
