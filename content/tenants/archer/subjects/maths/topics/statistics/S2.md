## Fluency (AO1)

1. A survey of 20 pupils' favourite colours gives: red 5, blue 8, green 4, yellow 3. Find the angle for red on a pie chart.
2. Using the data in Q1, find the angle for blue.
3. Using the data in Q1, find the angle for green.
4. Using the data in Q1, find the angle for yellow, then check that all four angles add to 360°.
5. A two-way table shows: of 30 pupils, 12 boys like football and 6 boys do not; 8 girls like football and 4 girls do not. How many boys are there in total?
6. Using the table in Q5, how many pupils in total like football?
7. A grouped frequency table for pupils' heights (cm) has class intervals $130 \leq h < 140$, $140 \leq h < 150$, $150 \leq h < 160$. Which interval would a pupil of height 145 cm belong to?
8. A grouped frequency table shows: $0 \leq t < 10$ minutes — 4 pupils, $10 \leq t < 20$ minutes — 9 pupils, $20 \leq t < 30$ minutes — 7 pupils. Find the total number of pupils.
9. A vertical line chart shows the number of pets owned by pupils: 0 pets — 6, 1 pet — 10, 2 pets — 5, 3 pets — 2. How many pupils were surveyed in total?
10. A time series table shows a shop's sales (£) each month: Jan 200, Feb 250, Mar 220, Apr 300. In which month were sales highest?
11. Using the time series data in Q10, find the increase in sales from January to April.
12. A pie chart shows a sector with angle 90° representing a category. What fraction of the total does this category represent?

## Reasoning (AO2)

13. Mia says a pie chart angle is found by dividing the total by the frequency. Explain why this is the wrong way round, and give the correct method.

14. **True or false?** In a two-way table, the row totals and column totals should both add up to the same overall total.

15. Ravi says a vertical line chart and a bar chart show exactly the same type of data in exactly the same way. Explain the difference between them.

16. **Always, sometimes, or never?** A time series graph is used to show how data changes over time.

17. Priya says that grouped frequency tables lose some detail compared with an ungrouped list of data. Explain why this is true, using an example.

18. Explain why the class intervals in a grouped frequency table (like $130 \leq h < 140$) must not overlap.

19. Sloan says a pie chart is better than a bar chart for showing exact values. Explain why a bar chart is usually better for reading off exact values.

20. Explain how you would decide, for a value exactly on the boundary (like 140 cm in a table with intervals $130 \leq h < 140$ and $140 \leq h < 150$), which interval it belongs to.

## Problem (AO3)

21. A survey of 60 pupils' travel to school gives: walk 24, car 15, bus 18, bike 3. Find the pie chart angle for each method.

22. A two-way table shows 50 pupils: 18 boys play an instrument, 12 boys do not; 14 girls play an instrument, 6 girls do not. Find the total number of pupils who play an instrument, and the total number of girls.

23. A grouped frequency table for exam scores shows: $0$–$20$ — 3 pupils, $21$–$40$ — 7 pupils, $41$–$60$ — 12 pupils, $61$–$80$ — 10 pupils, $81$–$100$ — 4 pupils. Find the total number of pupils, and the modal class (the class interval with the highest frequency).

24. A shop's monthly sales (£) over 6 months are: 300, 320, 280, 350, 400, 380. Describe the overall trend shown by this time series data.

25. A pie chart for 108 pupils' favourite lunch shows: sandwiches 150°, hot meal 120°, salad 90°. Find the number of pupils who chose each option.

26. A vertical line chart shows the number of siblings for 25 pupils: 0 — 5, 1 — 10, 2 — 6, 3 — 3, 4 — 1. Find the total number of pupils, and state which number of siblings is the mode.

## Stretch

27. A pie chart has 4 sectors with angles $x$, $2x$, $3x$, and $4x$. Find the value of $x$, then find each angle.

28. A survey of 144 people's favourite drink is shown on a pie chart with angles: tea 100°, coffee 140°, juice 80°, water 40°. Find the number of people who chose each drink.

29. Design a grouped frequency table for 40 pupils' times (in seconds) running 100 m, using class intervals of width 2 seconds, starting from 12 seconds. Choose sensible frequencies that add up to 40, and identify the modal class.

30. A time series shows a company's profits (£1000s) over 5 years: 20, 18, 25, 30, 28. Describe the overall trend, and explain one limitation of using only 5 data points to predict future profits.
