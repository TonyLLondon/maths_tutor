# Worksheet markdown

Place one worksheet per file: `{slug}.md`

- `slug` must match the filename (e.g. `factors-multiples-week1.md` → slug `factors-multiples-week1`).
- Use `topic` codes from the in-app topic catalog (N4, R1, …).
- Body supports GFM markdown and `$...$` maths via KaTeX.

Tenant `archer` is the default login profile for tutoring.
